В проекте использовались: 

* База данных HyperSQL 

* Веб-сервер Apache Tomcat 

Инструкция:
1. Скомпилировать класс Start для создания базы данных;
2. Подключить базу данных к стандартной утилите Intellij Idea (справа вкладка database, нажать '+' -> DataSource -> HPSQLDB), 
которая будет находится на сервере по пути apache-tomcat\bin\hyperSQL\travel с параметрами;
3. Запустить скрипты, находящиеся в файле create_tables, который лежит на гидхабе;
3. Запустить скрипты, находящиеся в файле test_data, который лежит на гидхабе;
4. Cкомпилировать, запаковать и задеплоить приложение на сервер.
